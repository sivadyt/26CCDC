============

Run EvtxECmd to make a readable CSV file *(Security, System, etc):
mkdir C:\temp
cd *\EvtxECmd
.\EvtxECmd.exe -f C:\Windows\System32\winevt\Logs\Security.evtx --csv "C:\temp\" --csvf inv.csv

run timeline explorer
import inv.csv

Re-run as needed to update logs in .csv file.
